# InfraQuest — 260 Bricks

A single-file interactive curriculum: 260 daily lessons (52 weeks, Mon–Fri) taking you from zero to job-ready Infrastructure / Cloud / Platform Engineer. Every brick on the wall is one day — tap it to see what to learn, exact tasks and commands, the proof to save, the beginner mistake to avoid, the small win, and a Quran/hadith quote.
